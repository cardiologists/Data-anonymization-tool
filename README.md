# Medical Data Anonymization Tool

A software tool designed to anonymize sensitive medical data by removing or masking personally identifiable information (PII/PHI) from medical datasets and medical images.  
The goal of this project is to support secure data sharing, research, and compliance with privacy regulations such as HIPAA.

---

## Features

- Anonymization of sensitive metadata and identifiers
- Support for medical imaging data (e.g., MRI, CT, DICOM)
- Removal or masking of patient-related fields
- Batch processing support
- Simple and user-friendly workflow
- Designed for research, clinical, and data engineering use cases

---

## Use Cases

- Preparing medical datasets for research and collaboration
- De-identifying imaging data before analysis or model training
- Reducing privacy risk when sharing clinical data
- Supporting regulatory compliance and data governance
